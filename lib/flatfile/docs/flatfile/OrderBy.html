		Creates a new OrderBy structure

The $compareAs parameter can be supplied using one of the pre-defined constants, but  this is actually implemented by defining the constants as names of functions to do the   comparison.  You can therefore supply the name of any function that works like  http://www.php.net/strcmp to implement custom ordering.
