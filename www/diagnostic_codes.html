This document contains a literal translation of the TapirLink diagnostic codes contained in the source XML file, tapir_errors.xml. This file is used to generate this document and the PHP code tapir_errorconst.php

Diagnostic Levels

Level Description
debug Message contains diagnostic information that can be helpful for determing problems with an installation. WARNING: content in the debug diagnostic can provide sensitive information about the provider installation.
info Message contains information about the status of the provider service or the recent operation.
warn Message contains information warning that the operation of the provider service may be incorrect, but no particular error was detected.
error An error occurred. The results that were generated should not be trusted as the content may be incorrect.
fatal Normal operation of the provider services was compromised and execution of the request could not be completed.

Diagnostic Codes

Value Level Description
DEBUG_MESSAGE debug Displays a debug message.
SYSTEM_MESSAGE info Displays a system message.
UNSUPPORTED_CAPABILITY warning Indicates that a certain capability is not supported.
UNSUPPORTED_SCHEMA_COMPONENT warning Indicates that an unknown/unsupported component was found in the response structure definition.
UNMAPPED_CONCEPT error Indicates that a certain concept is not mapped.
UNKNOWN_VARIABLE error Indicates that a certain environment variable is unknown.
UNSEARCHABLE_CONCEPT error Indicates that a certain concept used in the filter is not searchable.
INVALID_REQUEST error Indicates that the request is invalid.
TRUNCATED_RESPONSE warn Indicates that the request is invalid.
CONTENT_UNAVAILABLE warn Indicates that a concept is unmapped or an environment variable is unsupported.
UNSUPPORTED_OUTPUT_MODEL error Indicates that the output model provided is not supported.
RESPONSE_STRUCTURE_ISSUE error Indicates some problem in the response structure.
INVALID_FILTER error An invalid filter was provided in the request.
MISSING_PARAMETER error Indicates that a parameter is missing.
VERSION_MISMATCH warn Indicates a mismatch between the version of a software or component that is known to work with TapirLink and the current software or component installed.
IO_ERROR error Indicates an input/output error (reading from or writting to the file system).
XML_PARSE_ERROR error Indicates an error when parsing an XML string or file.
SERVER_SETUP_ERROR error Indicates an error in the server configuration.
DATABASE_CONNECTION_ERROR error There was a communication problem between the script and the database.
DATABASE_ERROR error Reports an error message that was generated by the database driver. The text of the message is the error message provided by the database interface.
INVALID_FILTER_TERM error A query term was not in the expected form for the searchable concept.
RESOURCE_NOT_FOUND error The requested resource was not available on the data provider.
CONFIGURATION_FAILURE fatal The configuration information for the provider could not be found or loaded, and hence the requested operation could not be completed.
CONFIGURATION_ISSUE warn A potential issue was detected in the configuration.
LOG_ERROR error Problem occured when opening or writing to a log file.
GENERAL_ERROR error A catch-all diagnostic for where no other appropriate category is available.
MISSING_LIBRARY warn The specified library is unavailable.
DURATION info A float value indicating the number of seconds to produce the response.
MEMORY_PEAK info Memory peak when processing response on the server.
DATA_VALIDATION_ERROR error A validation error occurred after submitting a form during configuration.
UDDI_ERROR error An error occurred during the UDDI registration process.
INTERNAL_ERROR error An error (probably during development) prevented the configurator to work properly.
